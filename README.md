# Linuxmit-miniscript
Script to minimized Linuxmint 19.3

## How to
* Download The package
* $ cd mini
* $ chmod +x run
* $ sudo ./run

## Clone
* $ git clone https://github.com/Litle-Panda/Linuxmit-miniscript.git
* $ chmod +x run
* $ sudo ./run
